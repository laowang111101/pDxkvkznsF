## 部署步骤

1. 克隆或下载项目源码
2. 安装依赖包
   ```bash
   pip install -r requirements.txt
   ```
3. 配置数据库
4. 运行迁移
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
5. 启动开发服务器
   ```bash
   python manage.py runserver
   ```
